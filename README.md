# Copilot Agent Demo

A simple Python calculator project demonstrating basic arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
from main import add, subtract, multiply

print(add(5, 3))        # 8
print(subtract(10, 4))  # 6
print(multiply(6, 7))   # 42
```

## Running the Script

```bash
python main.py
```
